# Customer Behaviour Analytics Platform

## 📌 Business Problem
Understanding customer spending patterns is critical for revenue growth and strategic targeting. 
This project analyzes transactional data to identify high-value customers, revenue concentration, and behavioral segments.

---

## 🛠 Tech Stack
- MySQL (Database Layer)
- SQL (Aggregations, Segmentation, KPI Modeling)
- Python (Pandas, NumPy, EDA)
- Power BI (Interactive Dashboards)

---

## 📊 Dataset
Transactional dataset containing customer purchases, payment modes, and revenue metrics.

---

## ⚙️ Project Workflow
1. Database design and data ingestion using MySQL
2. Data cleaning and transformation using SQL
3. Customer segmentation using revenue and behavioral metrics
4. Exploratory Data Analysis using Python
5. Dashboard visualization for stakeholder reporting

---

## 📈 Key Insights
- Identified top 20% customers contributing to majority revenue (Pareto principle)
- Segmented customers based on purchase frequency and spending behavior
- Highlighted revenue concentration risks
- Built KPI dashboard for monitoring revenue trends
